## OAuth
From [tweepy](https://github.com/tweepy/tweepy):

```
OAUTH_HOST='api.twitter.com' #REST API
OAUTH_ROOT='/oauth/'
```

1. **URL**:  `'https://' + OAUTH_HOST + OAUTH_ROOT` or `https://api.twitter.com/oauth/`
2. **Set consumer key/secret**
3. **Set acess token/secret**
